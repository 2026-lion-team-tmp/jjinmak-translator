# 기술 스택

## 프론트엔드 (client/)

- **React** — UI 렌더링, 슬롯머신 애니메이션, 페이지 전환
- **React Router** — 메인 페이지 / 결과물 페이지 라우팅

## 백엔드 (server/)

- **Node.js + Express** — API 서버
- **SQLite** — 이름별 돌린 횟수, 랭킹 데이터 저장

## 프로젝트 구조

```
jjinmak-translator/
├── client/          ← React 프론트엔드
│   └── src/
├── server/          ← Express API 서버
│   └── db/          ← SQLite DB 파일
├── docs/            ← 기획서, 기술 스택 등 문서
└── CLAUDE.md
```

## 통신 방식

- 프론트엔드 → 백엔드: REST API 호출
- 개발 시: React 개발 서버(포트 3000) + Express 서버(포트 5000) 동시 실행
- 배포 시: React 빌드 결과물을 Express가 정적 파일로 서빙
