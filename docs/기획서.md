# 찐막 속마음 번역기 기획

## 아이디어의 핵심 키워드

- 속마음
- 무작위
- 남 탓 & 억까
- 긁음
- 5초 짜리
- 시험 기간 딴 짓
- 이름 입력

## 페이지 구성

### 메인 페이지

```
(큰 글씨)우리 앱 이름

- 1등: 나지성 - [돌린 횟수]판 째 게임중!
- 2등: 차정은 - [돌린 횟수]판 째 게임중!
- 3등: 이도은 - [돌린 횟수]판 째 게임중!

(큰 글씨)친구 이름은...   [빈 칸]
(작은 글씨)랭크에 기록 안 할래요. [v]

     >> [ 속마음 번역하기 ] <<
```

### 결과물 페이지

```
(중간 글씨) (이름 강조)'나지성'의 속마음은…

(매우 큰 글씨 + 등장할 때 슬롯머신 연출) "이걸 더 안해??"
(작지만 강조된 글씨)현재 [돌린 횟수]판 째
    >> [공유하기] <<

[ 다시 돌리기 ] [ 돌아가기 ]
```
